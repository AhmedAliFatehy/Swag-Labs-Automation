# Swag Labs Automation Testing Project

This is an automation testing project built using **Selenium WebDriver, Java, TestNG, and Maven**.  
The framework follows the **Page Object Model (POM)** and reads test data from **JSON files**.

## Features Covered
- Automated **3 login scenarios** (valid, invalid, password required)
- Validated that all **6 products** appear on the inventory page
- Verified **cart visibility** and UI elements
- Checked **social media links**
- Added products to cart, verified items, and removed one successfully
- Completed the full checkout flow (Add product → Cart → Checkout Info → Overview → Finish Page → Order completion)

## Tools & Technologies
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- JSON (for test data)  
- Git / GitHub  

## How to Run
1. Clone the repository  
2. Install dependencies using Maven  
3. Run tests from TestNG.xml or through your IDE
