# Swag Labs Automation Testing Project

This is an automation testing project built using **Selenium WebDriver, Java, TestNG, and Maven**.  
The framework follows the **Page Object Model (POM)** and uses **JSON files** for external test data.

## Features Covered
- Automated **3 login scenarios**:
  - Valid login  
  - Invalid username/password  
  - Password required  
- Validated that all **6 products** appear on the inventory page  
- Verified **cart visibility** and UI elements  
- Checked all **social media footer links**  
- Added products to cart, verified items, and removed items successfully  
- **Completed the full checkout flow**  
  (Add product → Cart → Checkout Info → Overview → Finish Page → Order Completion)

## Tools & Technologies
- Java  
- Selenium WebDriver  
- TestNG  
- Maven  
- JSON (test data)  
- Git / GitHub  
- Page Object Model (POM)

## How to Run
1. Clone the repository.
2. Install dependencies using Maven.
3. Run tests through **testng.xml** or directly from your IDE.
